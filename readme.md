# Humble Bundle Key Collector

- create a key collection in the browser local storage so it persists.
- as keys are detected, unique keys will be added to the collection

# Steam Redeemer

- accepts keys in the form a json compatible array of product keys
- input keys are save to local storage, more keys can be added at a later time
- a seperate local storage collection is used to keep track of the state of each key as it's tested, and the script will resume with the last key that isn't detected to be 'owned'
- fill the key, check SSA and click continue automatically
- requires manually to move to the next key
